# Technology Stack: ON3 Athletics Admin Dashboard

## Frontend
- **Framework**: Next.js 14+ (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **UI Components**: shadcn/ui
- **Icons**: Lucide-react
- **Animations**: Framer Motion

## Backend & Infrastructure
- **Authentication**: Firebase Auth (Email/Password)
- **Database**: Cloud Firestore (Real-time `onSnapshot` listeners)
- **Hosting**: Vercel (Optimized for Next.js)

## Development & Tooling
- **State Management**: React Context / Hooks (Optimistic UI updates)
- **Data Fetching**: Firebase SDK
- **Environment Management**: `.env` (Strictly managed for Firebase keys)
